# Atividade Prática - Aplicação de WorkFlows com Git Actions

**Autor:** Maria Clara Cardoso Costa  
**Descrição:**  
O **Agenda Actions** foi criado para demonstrar, de forma prática e simples, como usar **GitHub Actions** para automatizar tarefas dentro de um repositório.  
Este projeto tem como objetivo demonstrar a aplicação de **Workflows com GitHub Actions**, utilizando **tags semânticas** (como `v1.0.0`, `v1.1.0`, `v2.0.0`, etc.) para automatizar o processo de criação de **releases**.
